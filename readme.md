# Цель проекта

Создать «нечто», что поможет при решении вопроса, связанного с выбором квартиры для ее дальнейшей покупки на основании данных из публичных источников.

# Какую проблему решаем

Большинство ресурсов по продаже/покупке недвижимости не предоставляют в общий доступ аналитических инструментов, которые могли бы помочь в решении вопроса, связанного с выбором недвижимости для ее дальнейшей покупки. В данном проекте решалось две основных задачи. Первая — это аккумуляция и агрегация данных с возможностью дальнейшей обработки. Вторая — реализация механизма, который поможет выбрать тот или иной вариант квартиры на основании простых, понятных, просто получаемых и легко интерпретируемых параметров.

## Критерии выбора квартиры
В общем виде было выделено три базовых составных критерия, на основании показателей которых осуществлялся выбор квартиры.

    - Кухня `(хорошо/плохо)`
    - Балкон `(хорошо/плохо)`
2. Качестно дома (R2)
    - Дата капремонта `(дата не ранее чем через 5 лет)`
    - Дата замены лифта `(дата не ранее чем через 5 лет)`

    - Наличие ступенек при входе в подъезд `(да/нет)`
3. Инфраструктура района. Шаговая доступность 500 метров (R3)
